# Movie-Ticket-Booking-System
A console-based Movie Ticket Booking System that allows users to view movies, select showtimes, choose seats, and book tickets through a simple console interface.

Description
This project is a command-line Movie Ticket Booking System developed in C for the CSC 1031 Group Project. The system allows users to view movie showtimes, book seats, cancel bookings, search bookings, and view revenue reports.

Seat Layout

- 5 Rows (A–E)
- 10 Seats per Row (1–10)
- Total Seats per Showtime: 50

Ticket Prices

- Rows A–B: Rs. 500 (Regular)
- Rows C–D: Rs. 750 (Premium)
- Row E: Rs. 1000 (VIP)

Features

1.View movie showtimes
2. View seat map
3.Book a seat
4. Cancel a booking
5. Search bookings
6. View revenue report
7.Exit

Requirements

- GCC Compiler
- C Programming Language

Compile

gcc main.c -o booking

Run

./booking

(Windows)

booking.exe
